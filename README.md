## 📝 Blog Posts

### Blog 1 — `blog-1.md`
**Stop Copy-Pasting Your Interfaces — Use `Pick` and `Omit` Instead**

Covers how `Pick` and `Omit` utility types help you create specialized types from a master interface without duplicating code — keeping your project DRY (Don't Repeat Yourself).

### Blog 2 — `blog-2.md`
**Why I Stopped Writing Separate Functions for Every Type — A Beginner's Guide to TypeScript Generics**

Covers how Generics allow you to write reusable, strictly-typed functions that work with any data type — without sacrificing type safety.

## 🚀 How to Run

Make sure you have [Node.js](https://nodejs.org/) and [TypeScript](https://www.typescriptlang.org/) installed.

```bash
# Install TypeScript globally (if not already installed)
npm install -g typescript

# Compile the solutions file
tsc solutions.ts

# Run the compiled output
node solutions.js
```
